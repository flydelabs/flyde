# Getting Started with Visual AI Flows

Welcome to Visual AI Flows! This guide will help you get started with integrating Visual AI Flows into your codebase.

## Installation

```bash
npm install @flyde/core @flyde/flow-editor
```

## Basic Setup

Here's how to set up Visual AI Flows in your project:

```typescript
import { createFlyde } from '@flyde/core';

// Initialize Flyde
const flyde = createFlyde();

// Create a simple workflow
const workflow = flyde.createWorkflow({
  name: 'Simple AI Workflow',
  nodes: [
    // Your workflow nodes will go here
  ]
});
```

## Next Steps

Once you have the basic setup, you can:

1. Create your first AI workflow
2. Integrate it with your existing codebase
3. Debug and visualize your workflow

Check out the [Core Concepts](/docs/concepts) section to learn more about how Visual AI Flows works.

## Example: Simple AI Agent

Here's a simple example of an AI agent workflow:

```typescript
const aiAgentWorkflow = flyde.createWorkflow({
  name: 'AI Agent',
  nodes: [
    {
      id: 'promptNode',
      type: 'Prompt',
      config: {
        text: 'What can I help you with today?'
      },
      position: { x: 100, y: 100 }
    },
    {
      id: 'llmNode',
      type: 'LLM',
      config: {
        model: 'gpt-4'
      },
      position: { x: 300, y: 100 }
    },
    {
      id: 'outputNode',
      type: 'Output',
      position: { x: 500, y: 100 }
    }
  ],
  edges: [
    {
      source: 'promptNode',
      target: 'llmNode'
    },
    {
      source: 'llmNode',
      target: 'outputNode'
    }
  ]
});
```

This is just a basic example. Visual AI Flows allows you to create much more complex workflows with multiple AI models, tools, and custom logic.
